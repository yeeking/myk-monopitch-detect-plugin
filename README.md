# Monophonic pitch tracking plugin translated from Supercollider's Pitch UGen

```
git clone https://github.com/yeeking/myk-monopitch-detect-plugin.git 
cd libs
git clone https://github.com/juce-framework/JUCE.git
cd ..
cmake -B build .
# change j to how many cores you have for max unixoid compile speed
cmake --build build --config Release -j 10
```
